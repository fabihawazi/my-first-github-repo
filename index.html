My First Blog

1 in a Life time set the user: Jump to the VS Code>Make a html file in VS Code(My first github repo folder)> Full html code> Give a title name> save it> Now click Terminal> New Terminal/click (ctrl+J)> click Powershell down arrow symbol> command prompt> Enter key> Copy from Github account/write git init> Enter key> write git add> click spacebar> click dot key> Enter key> write git commit -m “my first git”> Enter click> write the mail address> Enter> write the user name> Enter> (To check: git config -l> Enter key>WQ to go back)>Now click Up Arrow 2\3 times to get previous git commit> Enter click>Copy Paste Enter git branch from github to VS Code> Copy Paste Enter git remote> Copy Paste Enter git push> Sign in with your browser> Reload github account.
